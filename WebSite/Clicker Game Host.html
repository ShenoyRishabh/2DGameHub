Main Instructions:
The Sprite keeps increasing in size, it pops(you lose) if it touches the thorns placed beside it. Click the sprite to decrease its size and prevent it from popping.
Click - Decrease size of ball
Z - Hard mode
X - Easy mode
(Note: Click speed is only counted for the first ten seconds)
